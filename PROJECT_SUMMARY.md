# 教学实训智能体软件 - 项目总结

## 项目概述

本项目是一个基于开源大模型的教学实训智能体软件，旨在解决传统实训教学中的痛点问题，通过AI技术提升教学效率和学习效果。

## 核心功能

### 教师侧功能
1. **智能备课设计**
   - 基于课程大纲和知识库自动生成备课内容
   - 包含教学目标、重点难点、教学方法等
   - 支持时间分配和课后作业建议

2. **考核内容生成**
   - 自动生成多种类型的练习题
   - 支持单选题、多选题、填空题、简答题、编程题
   - 包含答案和详细解析

3. **学情数据分析**
   - 自动化检测学生答案
   - 提供错误定位与修正建议
   - 生成学习进度报告

### 学生侧功能
1. **在线学习助手**
   - 基于知识库的智能问答
   - 提供个性化学习指导
   - 支持编程代码示例

2. **实时练习评测**
   - 根据学习情况生成练习题
   - AI自动评估答案
   - 提供即时反馈和建议

### 管理侧功能
1. **用户管理**
   - 管理员、教师、学生角色管理
   - 用户权限控制

2. **课件资源管理**
   - 按学科管理课件和练习资源
   - 支持资源导出

3. **大屏数据概览**
   - 教师使用统计和活跃板块
   - 学生学习效果分析
   - 教学效率指数

## 技术架构

### 后端技术栈
- **语言**: Go 1.19+
- **Web框架**: Gin
- **ORM**: GORM
- **数据库**: MySQL 8.0+
- **缓存**: Redis 6.0+
- **认证**: JWT
- **AI集成**: OpenAI API

### 前端技术栈
- **框架**: Vue.js 3
- **状态管理**: Pinia
- **UI组件**: Element Plus
- **路由**: Vue Router
- **构建工具**: Vite
- **图表**: ECharts

### 数据库设计
包含以下核心表：
- users (用户表)
- courses (课程表)
- chapters (章节表)
- knowledge (知识点表)
- exercises (练习表)
- questions (题目表)
- student_answers (学生答案表)
- chat_sessions (聊天会话表)
- knowledge_base (知识库表)
